Keycloak Example

Keycloak is a user management tool, which allows you to handle user details, logins, logouts and verification of the token. You will see in this small tutorial, that once you have configured correctly keycloak, you are able to get the user data and the token generated. Going through this code, you will be able to see how to do this in Angular.
